Downloading Bloomberg data

  • We download data to construct a value score and momentum for all stocks in MSCI World.
# Define fields we want to retrieve
my_fields <- c("best_pe_ratio",
               "px_to_book_ratio",
               "last_close_trr_1yr",
               "last_close_trr_1mo")

# Download data from Bloomberg
df_blb_tmp <- bdp(securities = my_tickers, fields = my_fields)

# Add ticker to data frame
df_blb <- tibble(ticker = rownames(df_blb_tmp)) |> 
  bind_cols(df_blb_tmp) |> 
  as_tibble()

# Join Bloomberg data with input data
df <- inner_join(df_bm, df_blb, by = "ticker")

Distribution value metrics

Construct combined value score

  • Want to compute a value score based on both Price/Book and Price/Earnings.
  • Scale for Price/Book and Price/Earnings are different.
  • We can combine them by computing a Z-score \[ \begin{align} Z_{pe} &= \frac{x_{pe} - \mu_{pe}}{\sigma_{pe}} \\ Z_{pb} &= \frac{x_{pb} - \mu_{pb}}{\sigma_{pb}} \end{align} \]

Can compute weighed sum of z-scores to create composite value metric

\[ \text{Value} = w \times Z_{pe} + (1-w) \times Z_{pb} \]

Computing momentum

The academic version of momentum is the return from \(t-12\) months to \(t-1\) months.

\[ M = \frac{P_{t-1}}{P_{t-12}} = \frac{P_{t-1}}{P_{t}} \times \frac{P_{t}}{P_{t-12}} = \frac{R_{t-12,t}}{R_{t-1, t}} \]

Hence, we can compute momentum based on last 12 month return and last 1 month return.

df <- df |>
  mutate(mom = (1 + (last_close_trr_1yr/100))
         / (1 + (last_close_trr_1mo/100)) - 1)

Creating rank scores

df_scores <- df |>
  drop_na() |> 
  mutate(z_pe = scale(-best_pe_ratio, center = T, scale = T)[,1],
         z_pb =  scale(-px_to_book_ratio, center = T, scale = T)[,1],
         z_value =scale(z_pe + z_pb, center = T, scale = T)[,1],
         z_mom = scale(mom, center = T, scale = T)[,1],
         value = 100 * percent_rank(z_pe + z_pb),
         momentum = 100 * percent_rank(z_mom)
  )
